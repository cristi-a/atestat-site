Sortarea prin selecție - O(N2)

Sortarea prin selecție (Selection sort) se bazează pe următoare idee:

  • fie un vector v[] cu n elemente;
  • plasăm în v[0] cea mai mică valoare din vector;
  • plasăm în v[1] cea mai mică valoare rămasă;
  • ș.a.m.d.

O descriere a algoritmului este:

  • parcurgem vectorul cu indicele i
  • parcurgem cu indicele j elementele din dreapta lui v[i]
  • dacă elementele x[i] și x[j] nu sunt în ordinea dorită, le interschimbăm.

Observații:

  • În algoritmul de mai sus, pentru fiecare valoare a lui i, în v[i] se obține cea mai mică valoare dintre elementele cu indici i, i+1, ..., n; altfel spus, pentru fiecare i, în v[i] se selectează minimul dintre elementele i, i+1, ..., n
  • Metoda se mai numește sortare prin selecție directă, sortare prin selecție implicită sau sortare prin interschimbare.

Algoritm:

                
                    void sortareSelectie(int v[], int n) 
                    {
                        for(int i=1; i<=n; i++) 
                        {
                            for(int j=i+1; j<=n; j++) 
                            {
                                if(v[i] > v[j]) 
                                {
                                    int aux = v[i];
                                    v[i] = v[j];
                                    v[j] = aux;
                                }
                            }
                        }
                    }
