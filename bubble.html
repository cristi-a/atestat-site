Metoda bulelor - O(N2)

Cunoscută și sub numele Bubble sort, metoda bulelor se bazează pe următoarea idee:

  • fie un vector v[] cu n elemente;
  • parcurgem vectorul și pentru oricare două elemente învecinate care nu sunt în ordinea dorită, le interschimbăm valorile;
  • după o singură parcurgere, vectorul nu se v a sorta, dar putem repeta parcurgerea;
  • dacă la o parcurgere nu se face nicio interschimbare, vectorul este sortat

O reprezentare a algoritmului este:

  • cât timp vectorul nu este sortat:
  • parcurgem vectorul
  • dacă două elemente învecinate nu sjunt în ordinea dorită le interschimbăm și schimbăm presupunerea inițială

Algoritm:

                
                    void metodaBulelor(int v[], int n) 
                    {
                        bool sortat = true;
                        do 
                        {
                            sortat = true;
                            for(int i=1; i<=n; i++)
                            {
                                if(v[i] > v[i+1]) 
                                {
                                    int aux = v[i];
                                    v[i] = v[i+1];
                                    v[i+1] = aux;
                                    sortat = false;
                                }
                            }
                        }while(!sortat);
                    }
